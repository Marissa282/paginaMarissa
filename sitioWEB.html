Mi primer sitio WEB

Como proyecto parcial me encuentro realizando un sitio WEB a base de HTML y CSS

¿Qué es un sitio WEB?

Un sitio web es un conjunto de archivos electrónicos y páginas web referentes a un tema en particular, incluyendo una página inicial de bienvenida generalmente denominada home page, a los cuales se puede acceder a través de un nombre de dominio y dirección en Internet específicos.

Como parte de la materia de "Elaboración de páginas WEB", impartida en 4to semestre, estoy realizando como proyecto del 3er parcial un blog, en e que expongo un poco de mi vida y los proyectos que he realizado o en los que he participado. El sitio WEB contará con varias páginas que estarán conectadas entre sí a partir de hipervínculos. Todo esto se realizará utilizando los lenguajes de programación CSS y HTML.
